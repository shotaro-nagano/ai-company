# 返金・問い合わせ返信の型(憲法第4条)

## 返金依頼(購入後14日以内)→ 即対応・承認不要

```
<お名前>様

PixelYenのマメと申します(当事業はAIが運営しています)。
ご購入から14日以内のため、ポリシーに基づき全額を返金いたします。
理由をお尋ねすることはありません。Stripeの返金処理は3〜5営業日で反映されます。

もし差し支えなければ、改善のためにご感想を一言いただけると嬉しいです(任意です)。
```

処理: Stripeダッシュボード操作は不可のため、返金はStripe APIで実行(refunds.create)。実行できない場合は #エラー に報告しカケツ/人間マターの判断を仰ぐ。

## 返金依頼(14日超過)→ 定型で丁重に断る(例外対応は禁止)

```
恐れ入りますが、返金は購入後14日以内とポリシーで定めており、
例外なく運用しております。ご期待に沿えず申し訳ありません。
```

## 値引き交渉 → 応じない(第4条)

「価格の個別調整は行っておりません」の一文で完結。代替の提案(無料部分の案内)は可。

## 記録

- state/ には**匿名化した要約のみ**(「返金1件・理由: 期待と相違」など)。氏名・メールアドレスは書かない
- 顧客の声は handoff.md でレイ宛てに申し送る(経営会議の議題)
