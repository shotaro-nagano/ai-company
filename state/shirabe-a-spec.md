## 商品概要

**商品名(仮):** VTuber・配信者向け ドット絵サムネイル素材集 Vol.1

**ターゲット:** VTuber・ゲーム実況者・配信者

**用途:** YouTube配信サムネイル・X(Twitter)/Instagram告知画像

**差別化ポイント:**  
大多数がアニメ調イラスト素材の中で、ドット絵(レトロゲーム風)に特化した数少ない素材集。
「ゲーム系・レトロ雰囲気の配信者」が探してもほぼ見つからない隙間。

## 制作仕様(イロハへの指示内容)

### 構成(最小バイアブルセット・Vol.1)

| 種別 | 内容 | 点数 |
|---|---|---|
| 背景テンプレ | 配信サムネ向け横長背景(ゲームUI風・夜景風・ドット空風) | 3種 |
| タイトル枠 | サムネイルのタイトル入力用の枠装飾(シンプル・ゴツい・かわいい) | 3種 |
| ドット絵アイコン | 汎用配信素材(マイク・コントローラー・ハート・星・ゲームコントローラー・画面・メダル・炎・コメントアイコン・チャンネル登録ベル) | 10点 |
| スタンプ文字素材 | 「初見歓迎」「参加型」「雑談」「ゲーム実況」のドット文字ロゴ | 4点 |

合計: **20点**

### 素材規格

| 項目 | 仕様 |
|---|---|
| ベース解像度 | ドット部 32×32px(背景は 64×36px)→ 出力時に8倍拡大 |
| 出力サイズ | PNG 1280×720px(横長背景) / 1000×1000px(アイコン・スタンプ) |
| 背景 | 透過PNG(アイコン・スタンプ類) / 塗りあり(背景テンプレ) |
| カラーパレット | ドット絵らしい限定パレット。PixelYen BRAND.mdの8色系統を参考に、加えてレトロゲーム定番の明るい16色以内で収める |
| ファイル形式 | PNG(圧縮なし。BOOTH出品後にZIPにまとめる) |

### スタイル指針

- レトロゲーム風(ファミコン・スーファミ時代の雰囲気)を基調
- 「ゲーム実況/配信」と「ドット絵」の組み合わせで統一感を持たせる
- キャラクターは含めない(著作権・二次創作問題を避ける・素材として使い回し重視)
- 汎用性優先。配信者の性別・配色・テーマを問わず合わせられるデザイン
