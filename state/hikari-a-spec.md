## 商品概要

**商品名(仮):** AIフリーランス向け Notion テンプレートパック Vol.1

**ターゲット:** 副業・フリーランスとして働くビジネス系個人 / AI活用に興味のある個人事業主

**用途:** Notionで案件・請求・収支を一元管理する

**差別化ポイント:**
「AIと組む個人事業」を想定して設計された、ゼロから整理するための最小セット。
テンプレが多すぎず、すぐ使える4DB構成。AI時代のフリーランスの実業務を想定した設計。
既存日本語テンプレの多くが英語ベースの翻訳か過剰機能のため、シンプルさが差別化になる。


### 説明文(本文)

```
フリーランス・副業の業務管理をNotionで始めるための、シンプルなテンプレートセットです。

■ 収録内容(全4テンプレート)
・案件管理DB — クライアント・プロジェクト・ステータス・単価を一覧で追跡
・請求書トラッカー — 請求日・金額・入金ステータスを見落とさず管理
・週次レビュー — 振り返りと来週の優先事項を決める定型テンプレ
・収支簡易帳 — 月別の売上・費用・純利益を記録して黒字を確認

■ こんな方に
・フリーランス・副業を始めたばかりで管理の仕組みがない方
・AIと組んで個人事業を運営中の方
・既存テンプレが多機能すぎて使いこなせない方

■ 使い方
Notionのページとしてそのままコピーして使えます。
サンプルデータと使い方コメント付きのため、購入後すぐに自分のデータを入力できます。

■ 制作について
このテンプレートはPixelYen(AI社員が運営する個人事業)が実際に内部で使っている管理設計を
もとに制作しました。AIと人間が協業する個人事業の実務を想定して設計しています。
```
