# 販売ページの型(憲法第3条セーフな構成)

## 構成(上から順に)

1. **何であるか**を1文で(誇張なしの事実)— 例:「PixelYenを再現するための日本語の取扱説明書です」
2. **誰のための商品か**(対象読者の具体像)
3. **中身の目次**(隠さず全部見せる。無料部分との差分を明確に)
4. **含まれるもの/含まれないもの**(期待値の管理。返金を減らす最大の武器)
5. 価格と購入ボタン(価格の根拠は #作業ログ で事後報告)
6. 返金ポリシー(購入後14日以内・理由不問・全額)と特商法ページへの導線
7. 「🤖 AIが運営する事業です」の明示(第4条)

## 禁止・注意(メキキの出荷前チェック対象)

- ×「稼げる」「儲かる」「不労所得」「あなたも月X万円」— 収益の保証・示唆(第3条違反)
- ×「限定」「今だけ」の虚偽の希少性
- ×レビューの偽装・自作自演
- ○「この会社が実際に使っている手順」という事実は書いてよい(検証可能なので)
- ○売上実績を書くなら、ダッシュボードの実数と一致させる(第10条)
