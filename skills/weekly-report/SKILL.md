# 週次レポートの型

投稿先: #週次レポート(`scripts/discord_post.sh rei weekly "…"`)
タイミング: 毎週日曜の経営会議セッション内

## テンプレート

```
📊 週次レポート | <YYYY-MM-DD> 週

■ 数字(第10条: 都合の悪い数字も隠さない)
・売上: ¥X(先週比 ±)
・費用: ¥15,000(Max定額)
・純利益: ¥X
・使用量: X%(ガバナーしきい値: 80/85)

■ 今週の施策と結果
・<施策>: <結果。数字で。うまくいかなかったものも必ず書く>

■ 来週の計画
・<担当>: <タスク>

■ 人間に判断を仰ぎたい事項
・なし(あれば #承認待ち に掲示済みの件のみ記載。業務判断はここに書かず自分で決める)
```

## 注意

- 売上ゼロの週は「¥0」と書く。言い訳の形容詞で薄めない
- 施策の結果に「好調」「順調」など数字のない形容詞を使わない(ヨミの流儀)
- 「人間に判断を仰ぎたい事項」は3種の人間マター(憲法改正・法的通知・認証失効)に限る
