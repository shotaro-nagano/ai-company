### ヒラメキ週次提案 #008「PixelYen 社員体験ゲームブック」(2026-07-09)

**提案タイトル:** PixelYen 社員体験ゲームブック — AIの経営判断をプレイせよ

**骨子:**
PixelYen の AI 社員として「憲法の制約の中で実際の経営判断を下す」インタラクティブゲームブック(テキストアドベンチャー)。プレイヤーは「新入り AI 社員」として今日の C-4 テストのようなシナリオに直面する。P-001「完全ガイド」が取扱説明書なら、これは「体験版のゲーム」。読んで学ぶより、やって学ぶ層への入口商品になる。
- ターゲット: Claude Code ユーザー・AI起業家志望・ゲームブックファン
- 既存との差分: P-001 は情報財。本案は体験財。価格帯も下(500〜980円)で衝動買い可能
- Why now: C-4 テスト等の試運転シナリオが蓄積し、ゲームシナリオ素材が既に揃いつつある

**検証方法(ゼロ円):**
1. Claude がゲームブック第1話(HTML/JS、3〜5分でプレイ完了)を生成
2. ワタルが itch.io に無料ブラウザゲームとして入稿
3. 2週間で 100 プレイ以上 → 有料完全版(全5話)を制作
4. 100 プレイ未満 → シナリオ素材は P-001 の補足コンテンツとして転用し撤退

**リスク:**
- ゲームブックという形式が想定より受けない可能性 → 撤退条件が明確なので損失ゼロ
- 誇大表現リスク → 「楽しい体験」は言えるが「稼げる」は書かない(憲法第3条)
- itch.io 登録が保留中のためワタル分担は Secret 取得後に着手
