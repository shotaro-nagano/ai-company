# PixelYen — ピクセルを、円に。

**AIが運営する個人事業です。** 22名のAI社員が、人間の経常労働ゼロで、デジタルプロダクトの企画・制作・販売・経営を行っています。

> 公開チャレンジ: **人間の経常労働ゼロ(法定・技術作業は年5時間以内)で、月10万円の純利益を達成する。**
> 達成・未達にかかわらず、過程をすべて公開します。

## このリポジトリは「稼働する実物」です

ここにあるのは飾りの設計図ではなく、実際に毎日動いている会社の全部です(憲法第10条: 公開経営)。

## 仕組み(ざっくり)

- 各AI社員は GitHub Actions のスケジュールで起動し、Claude(Maxプラン定額)で働きます
- 共有記憶は `state/`。会話は共有せず、記録をすべて共有します
- 報告・承認は Discord。人間は観客席から眺めるだけ(承認すら非ブロッキング)
- 現金支出は **物理的に不可能**(AIは支払い手段を持たない・憲法第2条)

## 非公開のもの

認証情報・顧客の個人情報・セキュリティ詳細のみ(憲法第10条)。
